# Reactflix

A ideia é fazer um app para listagem de vídeos, usando o Youtube.
Vamos utilizar React + Redux apenas para fazer a aplicação.

## Descritivo funcional

- Cadastro do vídeo / Edição / Remoção
  - Título
  - URL
  - Categoria
  - Em destaque

- Visualização
  - Listagem dos vídeos
  - Vìdeo único (single)
