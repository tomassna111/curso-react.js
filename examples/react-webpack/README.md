dentro desse diretório você irá encontrar todo o código, separado por aula.

## Versão do Node.js

Nesse módulo nós usamos a versão 6 do Node.js. Para usar essa versão sem precisar desinstalar sua versão atual, você pode utilizar o NVM.

O NVM é um gerenciador de versões do Node.js. Você pode ter várias versões instaladas no mesmo computador.

Ele é bastante simples de mexer. Para saber como fazer a instalação e usar os comandos do NPM, acesse: https://github.com/creationix/nvm
